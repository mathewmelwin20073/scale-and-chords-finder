#  Scale & Chords Finder (Java)
A beginner project built during my first year of B.Tech CSE
- This is a simple terminal based music theory project that:
- Generates major scales
- Displays chords in a key
- Finds the relative minor
## Example
- Input:
- C
- Output:
- ======================
- SCALE AND CHORDS FINDER
- ======================
- Enter the root note:
- C
- C Major Scale:
- C->D->E->F->G->A->B->
- Chords in the key of C are:
- C major|D minor|E minor|F major|G major|A minor|B dim|
- Relative minor:
- A minor
## Concepts Used
- Arrays
- Loops
- User Input
- Music Theory Logic
- Modular Arithmetic
